# 🤰 PregCare – A Safe Space for Every Mother

## 💭 The Idea
Pregnancy is not just a medical journey.
It's emotional, confusing, overwhelming… and sometimes lonely.

This project is a small attempt to create a **safe digital space** where women can:

* Seek guidance
* Feel understood
* Access basic health information
* And most importantly… not feel alone
---

## 🌸 What I Built
A static web interface that includes:

* 🩺 Pregnancy-related guidance sections
* 💬 Emotional support ("Meltdown?" safe space concept)
* 📊 Baby growth & vaccination awareness ideas
* 🤰 "Tip of the Day" feature for daily guidance
* 🧾 Modal-based "About Us" section
---

## ⚙️ Tech Used

* HTML
* CSS
* JavaScript
---

## 🚧 Current Limitations

* Buttons are not yet connected to pages
* No backend or data storage
* Static content only
---

## 🚀 Future Improvements
I want to take this much further:

* 🔗 Connect each section to detailed pages
* 🧠 Add AI-based chatbot for emotional support
* 📊 Build baby growth tracker with real data
* 🔔 Reminder system for checkups & vaccinations
* 👩‍⚕️ Integrate verified medical information
* 🌐 Make it accessible for rural users
---

## 💡 Why This Matters
Not every woman has:

* Immediate access to doctors
* Emotional support
* Reliable information

This platform aims to become a **companion during one of the most important phases of life**.
